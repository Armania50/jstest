Eindopdracht

Maak een rekenmachine in Javascript: 10 knopjes voor de getallen 0-9, 4 knopjes voor / * + - en 1 knopje voor "=". Tot slot een input waarin het resultaat wordt neergezet. Het programma werkt zo.

  • Druk op een getal button, bijvoorbeeld 8
  • Javascript onthoudt "8" in array "waarden", die bevat nu [8]
  • Druk op een operator (+ - / *), bijvoorbeeld +
  • De gekozen operator wordt onthouden in de variabele "operator", die bevat nu '+'
  • Druk op een tweede getal, bijvoorbeeld 3.
  • Javascript onthoudt "3" in array "waarden", die bevat nu [8,3]
  • Klik op =
  • Javascript neemt de twee getallen 8 en 3, kijkt welke operator uitgevoerd moet worden, doet de benodigde berekening, print de uitkomst naar de input 'resultaat' en maakt 'waarden' en 'operator' weer leeg.
